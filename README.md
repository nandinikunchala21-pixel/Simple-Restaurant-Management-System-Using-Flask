# 🍽️ Simple Restaurant Management System Using Flask

##  Overview
The **Simple Restaurant Management System** is a web-based application developed using the **Flask Framework** in Python. It helps restaurant owners or managers manage menu items, customer orders, and staff information through a simple and user-friendly interface.

This project does **not** use an external database. Instead, data is stored within the application, making it lightweight and easy to set up for learning purposes and small-scale restaurant management.


## Features
- Add, view, update, and delete menu items
- Manage customer orders
- Store and manage staff information
- Perform CRUD (Create, Read, Update, Delete) operations
- User-friendly interface built with Flask and HTML
- Basic input validation and error handling


## Technologies Used
- Python
- Flask
- HTML
- CSS
- Jinja2 Templates


##  Project Objectives
- Develop a restaurant management system using Flask.
- Implement CRUD operations without an external database.
- Create an easy-to-use web interface.
- Demonstrate Flask routing, templates, and form handling.
